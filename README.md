# 자동차 경주 게임

## 기능 구현

### Main Class

#### public Method

**main Method**



----

### Car Class

#### public Method

**getName Method**

- 자동차 이름을 반환한다.

**getPosition Method**

- 자동차 위치를 반환한다.

**increase_one_position Method**

- 자동차 위치를 1 증가시킨다.

---

### Manager Class

#### private Method

**generate_random_number Method**

- 0~9 사이의 랜덤 숫자를 생성한다.

**input_try_number Method**

- 시도할 회수를 입력한다.

**input_car_name Method**

- 자동차 이름을 입력하고 올바른 이름인지 검사한다.

**check_car_name**

- 입력된 자동차 이름이 올바른지 검사한다.

**generate_cars**

- 자동차 인스턴스를 생성한다.



#### public Method

**cars_go_to_stop**

- 생성된 랜덤 숫자에 따라 자동차가 전진할지 멈출지 결정한다.

**getNumber**

- 시도할 회수를 반환한다.

**print_winner Method**

- 승리자를 출력한다.

**check_go_or_stop Method**

- 랜덤으로 생성한 숫자의 범위를 검사한다.
